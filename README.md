# Sileo Repository

This is a custom Sileo repository containing various tweaks and packages for jailbroken iOS devices.

## Adding this Repository

1. Open Sileo
2. Go to Sources
3. Tap the + button
4. Add the following URL:
```
https://dora.github.io
```
